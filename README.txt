******************************************
ORBITZ - The Solar Explorer
******************************************

*INNAN KÖRNING*

* API SOM BEHÖVS FINNS BIFOGAD I KOD MAPPEN (json-simple 1.1.1)

* För att köra programmet så behöver man lägga till json-simple.1.1.1 i ett biblotek
	Se instruktioner hur detta göras i InteliJ (https://www.jetbrains.com/help/idea/library.html)

* Java 11 krävs för att köra programmet

* Java 11 inkluderar Java FX som används i Orbitz.

*UNDER KÖRNING*

* Det går att zooma genom att scrolla med musljulet (om man har mus).
	Har man en musplatta görs detta med genom att dra med två fingrar.
* Man kan trycka och dra för att förflytta sig i omgivningen.

* Tryck på en planet för att få upp en informationsfönster om planeten
